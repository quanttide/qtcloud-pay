# ROADMAP — 卖课支付

只做一件事：买家下单买课 → 付款 → 收到通知 → 查询确认。

## Phase 1 — 能用

- [x] `Provider` 接口定义
- [x] 微信支付 JSAPI（公众号/小程序卖课最常用）
- [x] 支付宝网页支付（PC 端卖课用）
- [x] 异步通知接收与签名验证
- [x] 订单查询
- [ ] 接入示例与文档

## Phase 2 — 够用

- [ ] 退款（学员申请退款）
- [ ] 错误处理与重试
- [ ] 基础测试

## 不做

- 分期付款
- 合单支付
- 跨境支付
- 订阅扣款
- 多渠道路由
- 对账系统
- 监控告警
